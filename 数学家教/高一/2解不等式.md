# 一元二次不等式
**基本形式**
$$\large ax^2+bx+c>0$$
**要点**
找零点，画图象
**例子**
$$\large x^2-5x-6<0,\quad 2x^2+x-6\geq0$$
# 分式不等式
**基本形式**
$$\large \dfrac{ax+b}{cx+d}>0$$
**要点**
分式化乘积，分母不为零
**例子**
$$\large \dfrac{4x+3}{1-x}\geq0,\quad \dfrac{2x+5}{x+1}\leq1$$
# 多项式、多项分式不等式
**例子**
$$\large x^4-4x^3-3x^2+14x-8\geq0,\quad \dfrac{x^2-59}{3x-1}\leq-1,\quad x^3+2x^2+4x>0$$
# 含有绝对值的不等式
**基本形式**
$$\large |x|>m,\quad |a|>|b|$$
**例子**
$$\large |x-1|>2,\quad |x+3|>1-2x,\quad |x-1|>|2x+3|$$

# 含有根式的不等式
**例子**
$$\large \sqrt{3-x}<x-1$$
**要点**
两边平方、自然定义域
