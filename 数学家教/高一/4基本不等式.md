# 基本不等式
$$\large \forall x,y>0\quad x+y\geq2\sqrt{xy}\quad当且仅当x=y取等$$
**要点**
和定值、积定值
**例题**
1. 实数$x,y$满足$xy=1$，求$2x^2+3y^2$最小值
2. 正数$a,b$满足$ab=4a+b+12$，求$ab$最小值
3. 实数$x,y$满足$2x+y=2$，求$9^x+3^y$最小值
4. 求$2x+\dfrac{1}{x-1}(x>1)$最小值
5. （符号）$x<4$，求$2x+\dfrac{1}{x-4}$最大值
6. $0<a<\sqrt2$，求$a\sqrt{2-a^2}$最大值
7. （换元）$a,b$为正实数，求$\dfrac{a}{a+2b}+\dfrac{b}{a+b}$最小值
8. （1代换）$x>0,y>0,x+y=1$，求$\dfrac{1}{2x}+\dfrac{x}{y+1}$最小值
9. （1代换）$\dfrac{1}{a}+\dfrac{2}{b}=2$，求$a+2b$最小值
10. （1代换）$0<x<3$，求$\dfrac{1}{x}+\dfrac{12}{3-x}$最小值
11. （1代换）$a>-1$，$b>0$，$a+2b=1$，求$\dfrac{2}{a+1}+\dfrac{1}{b}$最小值
12. （1代换）$a>0$，$b>0$，$\dfrac{2}{a+b}+\dfrac{1}{b}=1$，求$a+2b$最小值
13. （换元）$x>1$，$y>0$，$x^2-3x+y=0$，求$\dfrac{4}{x-1}+\dfrac{x}{y}$最小值
14. （非齐次）$2<x<4$，求$\dfrac{x^2}{x-2}+\dfrac{x^2}{4-x}$最小值
15. （非齐次）$x>0$，求$\dfrac{2x}{x^2-x+4}$最大值
16. （非齐次）$x>0$，求$\dfrac{3x^2+3x}{(\dfrac{3}{2}x^2+14)(14k^2+\dfrac{3}{2})}$最大值
17. （非齐次凑齐次、1代换）$x>0,y>0,x+y=1$，求$\dfrac{2x^2+x+1}{xy}$最小值
18. （消元）$x>0,y>0,x^2+xy-3=0$，求$4x+y$最小值
19. （换元）$x(x+y)=2+2y^2$，求$7x^2-y^2$最小值
20. （因式分解）$a>1,b>1$，$\dfrac{1}{a}+\dfrac{1}{b}=1$，求$\dfrac{4}{a-1}+\dfrac{9}{b-1}$最小值
21. （因式分解）$a>\dfrac{1}{4},b>\dfrac{1}{2}$，$\dfrac{1}{a}+\dfrac{2}{b}=4$，求$\dfrac{4a}{4a-1}+\dfrac{2b}{2b-1}$最小值
22. （多次放缩）$a>0,b>0$，求$\dfrac{1}{a}+\dfrac{a}{b^2}+b$最小值
23. （多放缩）$ab>0$，求$\dfrac{a^4+4b^4+1}{ab}$最小值
24. （多放缩）$xy>y^3>0$，求$xy+\dfrac{1}{y(x-y^2)}+\dfrac{1}{y^3}$最小值
# 不等式链
$$\forall x_1,...,x_n>0\quad\large \dfrac{n}{\dfrac{1}{x_1}+\cdots+\dfrac{1}{x_n}}\leq\sqrt[n]{x_1\cdots x_n}\leq\dfrac{x_1+\cdots+x_n}{n}\leq\sqrt{\dfrac{x_1^2+\cdots+x_n^2}{n}}\quad当且仅当x_1=\cdots=x_n时取等$$
## 1. 简单题：分式的最小化（高考频考点）
**题目**：已知实数 $x, y > 0$，且 $x + y = 2$，求 $\dfrac{1}{x} + \dfrac{1}{y}$ 的最小值。
**解析**：
> **利用不等式**：调和平均 $\leq$ 算术平均，即 $\dfrac{2}{\dfrac{1}{x} + \dfrac{1}{y}} \leq \dfrac{x + y}{2}$。
> 代入已知条件 $x + y = 2$，得：$\dfrac{2}{\dfrac{1}{x} + \dfrac{1}{y}} \leq \dfrac{2}{2} = 1$。
> 变形即得：$\dfrac{1}{x} + \dfrac{1}{y} \geq 2$。
> **取等条件**：当且仅当 $x = y = 1$ 时等号成立。故最小值为2。
> *(注：此题也常使用“乘1法”解决，但直接套用调和-算术不等式最迅速)*

## 2. 难题：经典的内斯比特不等式 (Nesbitt's Inequality)
**题目**：已知 $a, b, c > 0$，求证：$\dfrac{a}{b+c} + \dfrac{b}{c+a} + \dfrac{c}{a+b} \geq \dfrac{3}{2}$。
**解析**：
> **构造与利用不等式**：这道题硬凑很难，但如果利用**调和平均 $\leq$ 算术平均**，过程会极为优美。
> 我们在每个式子加上1：
> $\left(\dfrac{a}{b+c} + 1\right) + \left(\dfrac{b}{c+a} + 1\right) + \left(\dfrac{c}{a+b} + 1\right)$
> $= \dfrac{a+b+c}{b+c} + \dfrac{a+b+c}{c+a} + \dfrac{a+b+c}{a+b} = (a+b+c)\left(\dfrac{1}{b+c} + \dfrac{1}{c+a} + \dfrac{1}{a+b}\right)$
> 现在令 $x_1 = b+c, x_2 = c+a, x_3 = a+b$，注意 $x_1+x_2+x_3 = 2(a+b+c)$。
> 根据不等式链 $\dfrac{3}{\dfrac{1}{x_1} + \dfrac{1}{x_2} + \dfrac{1}{x_3}} \leq \dfrac{x_1+x_2+x_3}{3}$，变形可得：
> $(x_1+x_2+x_3)\left(\dfrac{1}{x_1} + \dfrac{1}{x_2} + \dfrac{1}{x_3}\right) \geq 9$  **(这就是著名的“和与倒数和的积 $\geq n^2$”结论)**
> 将 $x_1, x_2, x_3$ 代回，得：
> $2(a+b+c)\left(\dfrac{1}{b+c} + \dfrac{1}{c+a} + \dfrac{1}{a+b}\right) \geq 9$
> $(a+b+c)\left(\dfrac{1}{b+c} + \dfrac{1}{c+a} + \dfrac{1}{a+b}\right) \geq \dfrac{9}{2}$
> 将刚才多加的“3”减去：原文左边 $\geq \dfrac{9}{2} - 3 = \dfrac{3}{2}$。证明完毕！取等条件为 $a=b=c$。

# 柯西不等式
$$\large \forall x_1,y_1,x_2,y_2\in\mathbb{R}\quad (x_1^2+y_1^2)(x_2^2+y_2^2)\geq(x_1x_2+y_1y_2)^2\quad当且仅当x_1y_2=x_2y_1时取等$$
## 1. 简单题：圆上的最值问题
**题目**：已知实数 $x, y$ 满足 $x^2 + y^2 = 4$，求 $3x + 4y$ 的最大值与最小值。
**解析**：
> **利用不等式**：直接套用二维柯西不等式 $(x_1^2+y_1^2)(x_2^2+y_2^2) \ge (x_1x_2+y_1y_2)^2$。
> 构造：将 $3x+4y$ 视为 $(3\cdot x + 4\cdot y)$。
> 则 $(3^2 + 4^2)(x^2 + y^2) \geq (3x + 4y)^2$。
> 代入已知可得：$(25)(4) = 100 \geq (3x + 4y)^2$。
> 解得：$-10 \leq 3x + 4y \leq 10$。
> **取等条件**：当 $\dfrac{x}{3} = \dfrac{y}{4}$ 且 $x^2+y^2=4$ 时，分别取到最大值(x=6/5, y=8/5)和最小值。

## 2. 难题：无理函数的极值配凑
**题目**：已知实数 $x \in [2, 7]$，求函数 $f(x) = 2\sqrt{x-2} + 3\sqrt{7-x}$ 的最大值。
**解析**：
> **难点分析**：如果要用导数求极值，计算量极大，且涉及复杂的无理根号方程。但观察根号内部 $(x-2) + (7-x) = 5$ 是一常数！这正是柯西不等式的用武之地。
> **利用不等式**：令 $x_1 = 2, y_1 = 3$；$x_2 = \sqrt{x-2}, y_2 = \sqrt{7-x}$。
> 根据柯西不等式 $(x_1^2 + y_1^2)(x_2^2 + y_2^2) \geq (x_1x_2 + y_1y_2)^2$：
> $(2^2 + 3^2)\left[ (\sqrt{x-2})^2 + (\sqrt{7-x})^2 \right] \geq \left( 2\sqrt{x-2} + 3\sqrt{7-x} \right)^2$
> 即：$(4 + 9)(x - 2 + 7 - x) \geq [f(x)]^2$
> $13 \times 5 = 65 \geq [f(x)]^2$。
> 因为 $f(x) > 0$，所以 $f(x) \leq \sqrt{65}$。
> **验证取等条件**（很重要，否则可能是假最值）：当仅当 $\dfrac{\sqrt{x-2}}{2} = \dfrac{\sqrt{7-x}}{3}$ 时等号成立。平方得 $\dfrac{x-2}{4} = \dfrac{7-x}{9}$，解得 $x = \dfrac{46}{13}$。由于 $2 \leq \dfrac{46}{13} \leq 7$，最大值 $\sqrt{65}$ 可以取到。
# 三角不等式
$$\large\forall a,b\in\mathbb{R}\quad||a|-|b||\leq|a\pm b|\leq|a|+|b|\quad中间为+时，左侧异号取等，右侧同号取等$$
## 1. 简单题：求绝对值函数的最值
**题目**：求函数 $f(x) = |x - 2| + |x - 5|$ 的最小值。
**解析**：
> **利用不等式**：使用右侧的 $|a| + |b| \geq |a \pm b|$，为了消去变量 $x$，将内部一正一负相加。
> $f(x) = |x - 2| + |5 - x|$（绝对值内部可变号）
> $\ge |(x - 2) + (5 - x)| = |3| = 3$。
> **取等条件**：不等号中间为加号，故 $|a|+|b|=|a+b|$ 成立的条件是 $a,b$ 同号，即 $(x-2)(5-x) \geq 0$，解得 $2 \leq x \leq 5$。在此区间上，最小值恒为 3。

## 2. 难题：泛函结构下的放缩证明
**题目**：对于任意实数 $a, b$，求证：$\dfrac{|a+b|}{1+|a+b|} \leq \dfrac{|a|}{1+|a|} + \dfrac{|b|}{1+|b|}$。
*(此题也是大学数学分析中的经典距离度量空间证明)*
**解析**：
> **构造函数与利用不等式**：
> 设函数 $g(t) = \dfrac{t}{1+t} = 1 - \dfrac{1}{1+t}$。易知当 $t \ge 0$ 时，函数 $g(t)$ 是**严格单调递增**的。
> 根据绝对值三角不等式：$|a+b| \leq |a| + |b|$。
> 利用 $g(t)$ 的单调递增性，可得：
> $g(|a+b|) \leq g(|a|+|b|)$
> 即 $\dfrac{|a+b|}{1+|a+b|} \leq \dfrac{|a|+|b|}{1+|a|+|b|}$
> 接着，将右边拆成两项：
> $\dfrac{|a|+|b|}{1+|a|+|b|} = \dfrac{|a|}{1+|a|+|b|} + \dfrac{|b|}{1+|a|+|b|}$
> 因为分母 $1+|a|+|b| \ge 1+|a|$ 且 $1+|a|+|b| \ge 1+|b|$ （分母变小，分式变大），所以：
> $\dfrac{|a|}{1+|a|+|b|} \leq \dfrac{|a|}{1+|a|}$
> $\dfrac{|b|}{1+|a|+|b|} \leq \dfrac{|b|}{1+|b|}$
> 将两式相加，即可得到最终结论：$\dfrac{|a+b|}{1+|a+b|} \leq \dfrac{|a|}{1+|a|} + \dfrac{|b|}{1+|b|}$。
