# 要点
利用二次函数特征参数，限制函数图像，解不等式组
1. 韦达定理
$$\large x_1+x_2=-\dfrac{b}{a},\quad x_1x_2=\dfrac{c}{a}$$
2. 对称轴$$\large x=-\dfrac{b}{2a}$$
3. 判别式$$\large\Delta=b^2-4ac$$
4. 顶点$$\large P(-\dfrac{b}{2a},f(-\dfrac{b}{2a}))$$
5. 零点$$\large f(x)=a(x-x_1)(x-x_2)$$
6. 特殊点$$\large f(x_0)>y_0$$
# 例子
1. 方程$x^2-mx-m+3=0$有两个不等的实根，一个根大于零，一个根小于零
2. 方程$x^2-mx-m+3=0$有两个不等的实根$x_1,x_2$，满足$-1<x_1<x_2<2$
3. 方程$x^2+(m-2)x+5-m=0$有两个不等的实根，且均大于$2$
4. 方程$mx^2+2(m+3)x+2m+14=0$有两个不等实根，一个大于$4$，一个小于$4$
5. 方程$x^2+2(m-1)x+2m+6=0$有两个不等实根，满足$0<x_1<1<x_2<4$
6. （二次）方程$mx^2+(2m-3)x+4=0$只有一个正根且小于$1$
7. $x_1,x_2$是方程$4x^2-4mx+m+2$的两个实根，求$x_1^2+x_2^2$的最小值
