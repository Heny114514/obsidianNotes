
# 寄存器
## x86-32通用寄存器
- 8个32位寄存器：eax, ebx, ecx, edx, esi, edi, esp, ebp
- 8个16位寄存器：ax, bx, cx, dx, si, di, sp, bp
> 均为对应32位寄存器的低16位
- 8个8位寄存器：ah, al, bh, bl, ch, cl, dh, dl
> 分别为ax-dx的高8位和低9位，h-high，l-low

## x86-64通用寄存器
- 16个64位寄存器：rax, rbx, rcx, rdx, rbp, rsi, rdi, rsp, r8, r9, …, r15
- 16个32位寄存器：eax, ebx, …, esp, r8d, …, r15d
> 分别为对应64位寄存器的低双字(dword)
- 16个16位寄存器：ax, …, sp, r8w, …, r15w
> 分别为对应64位寄存器的最低字(word)
- 20个8位寄存器：al, bl, cl, dl, bpl, sip, dil, spl, r8b, …, r15b, ah, bh, ch, dh
> 前16个寄存器为对应64位寄存器的最低字节(byte)，ah-dh和x86-32相同
> 不能同时使用al-ah, …, dl-dh

## 标志寄存器
### SF(Sign Flag)
运算结果的最高二进制位为1，则SF=1，否则SF=0
### CF(Carry Flag)
运算时最高位向前进位，则CF=1，否则CF=0
### ZF(Zero Flag)
运算结果为0，则ZF=1，否则ZF=0
### OF(Overflow Flag)
加法执行前加数最高二进制位相同，且与结果最高位相反，则OF=1，否则OF=0

# 寻址方式

## 立即寻址（Immediate Addressing）

### 概念

操作数 **直接包含在指令中**，CPU 不需要再访问寄存器或内存。

### 示例

```asm
mov rax, 5
add rbx, 10
```

### 说明

- 操作数是常数
    
- 不需要访问内存
    
- 执行速度快
    


## 寄存器寻址（Register Addressing）

### 概念

操作数 **存放在寄存器中**。

### 示例

```asm
mov rax, rbx
add rax, rcx
```

### 说明

CPU 直接从寄存器读取数据。


### 直接寻址（Direct Addressing）

### 概念

**指令中直接给出操作数所在的内存地址**。

### 示例

```asm
mov rax, [0x600000]
```

### 地址计算

```
EA = 指令中的地址
```

EA = Effective Address（有效地址）

## 寄存器间接寻址（Register Indirect Addressing）

### 概念

**寄存器中存放的是内存地址**。

CPU 通过寄存器中的地址访问内存。

### 示例

```asm
mov rax, [rbx]
```

### 地址计算

```
EA = (RBX)
```

含义：

```
rax = Memory[rbx]
```


## 基址寻址（Base Addressing）

### 概念

使用 **基址寄存器 + 偏移量** 来计算内存地址。

### 地址计算

```
EA = Base + Displacement
```

### 示例

```asm
mov rax, [rbx + 8]
```

### 含义

```
rax = Memory[rbx + 8]
```

### 使用场景

常用于：

- 结构体访问
    
- 栈变量访问
    
- 数据结构
    

## 变址寻址（Indexed Addressing）

### 概念

使用 **变址寄存器 + 位移量** 来计算地址。

### 地址计算

```
EA = Index + Displacement
```

### 示例

```asm
mov rax, [rsi + 16]
```

### 使用场景

主要用于 **数组访问**。

例如：

```c
a[i]
```


## 基址变址寻址（Base + Index Addressing）

### 概念

地址由 **基址寄存器和变址寄存器共同计算**。

### 地址计算

```
EA = Base + Index × Scale + Displacement
```

其中

|参数|含义|
|---|---|
|Base|基址寄存器|
|Index|变址寄存器|
|Scale|比例因子 (1,2,4,8)|
|Displacement|位移|

### 示例

```asm
mov rax, [rbx + rsi*4 + 8]
```

### 说明

```
rax = Memory[rbx + rsi*4 + 8]
```

### 典型用途

数组访问：

```c
int a[10];
a[i]
```

如果 `int = 4字节`

```
地址 = base + i × 4
```


## x86-64 内存寻址统一公式

x86 架构的内存地址几乎都可以表示为：

```
EA = Base + Index × Scale + Displacement
```

说明：

| 元素           | 是否必须    |
| ------------ | ------- |
| Base         | 可选      |
| Index        | 可选      |
| Scale        | 1/2/4/8 |
| Displacement | 可选      |

示例：

```
[rax]
[rax+8]
[rax+rbx]
[rax+rbx*4]
[rax+rbx*4+8]
```


## 总结表

| 寻址方式    | 示例              | 地址计算                  |
| ------- | --------------- | --------------------- |
| 立即寻址    | `mov rax,5`     | 操作数在指令中               |
| 寄存器寻址   | `mov rax,rbx`   | 操作数在寄存器               |
| 直接寻址    | `[0x600000]`    | EA=Address            |
| 寄存器间接寻址 | `[rbx]`         | EA=(RBX)              |
| 基址寻址    | `[rbx+8]`       | Base+Disp             |
| 变址寻址    | `[rsi+16]`      | Index+Disp            |
| 基址变址寻址  | `[rbx+rsi*4+8]` | Base+Index×Scale+Disp |
# 常用机器指令

## 数据传送指令

### 一般传送
`mov opd, ops`
ops为源，opd为目的

### 有符号数传送
`movsx r16/r32, ops/非立即数`
传送时在高位补足符号位

### 无符号数传送
`movzx r16/r32, ops/非立即数`
传送时在高位补足0

### 传送偏移地址
`lea r32, m32`

`lea ecx, [eax+ebx*8+0ch] //将eax+ebx*8+0ch赋给ecx`


### 进栈
`push ops`
栈顶在低地址，进栈时栈寄存器先减再传送
```
//字入栈
esp-2 -> esp
字数据 -> [esp]

//双字入栈
esp-4 -> esp
双字 -> [esp]
```
### 出栈
`pop opd`

```
[esp] -> opd
esp+2/4 -> esp
```

## 算术运算指令

### 加法
```
//加1
inc opd:
opd+1 -> opd

//加
add opd, ops:
opd+ops -> opd

//带进位加
adc opd, ops:
opd+ops+cf -> opd
```

### 减法
```
//减1
dec opd:
opd-1 -> opd

//求补
neg opd:
~opd+1 -> opd

//减
sub opd, ops:
opd-ops -> opd

//带借位减
sbb opd, ops:
opd-ops-cf -> opd
```

### 乘法
opd与ops类型相同
```
//有符号
imul opd, ops:
opd*ops -> opd

imul opd, ops, n:
ops*n -> opd

imul ops:
字节: al*ops -> ax
字: ax*ops -> dx,ax
双字: eax*ops -> edx, eax

//无符号
mul ops:
字节: al*ops -> ax
字: ax*ops -> dx,ax
双字: eax*ops -> edx, eax
```

### 除法与符号扩展
```
//有符号
idiv ops
//无符号
div ops

字节: ax/ops -> al(商),ah(余)
字: (dx,ax)/ops -> ax(商),dx(余)
双字: (edx,eax)/ops -> eax(商),edx(余)

//字节转字
cbw: al中符号扩展至ah

//字转双字
cwd: ax符号扩展至dx

cwde: ax有符号数扩展为32位传送至eax

//双字转四字
cdq: eax扩展为64位送至(edx,eax)
```

### 按位运算
```
//求反
not opd:
~opd -> opd

//按位与
and opd, ops:
opd&ops -> opd

//按位或
or opd, ops:
opd|ops -> opd

//按位异或
xor opd, ops:
opd^ops -> opd

//测试指令（只影响标志位）
test opd, ops:
opd&ops
```

### 移位
```
语句格式： 操作符 opd, n/cl
只允许使用立即数或cl寄存器

//算数左移 shift arithmetic left
sal opd, n

//逻辑左移 shift logical left
shl opd, n

//算数右移
sar opd, n

//逻辑右移
shr opd, n
```
