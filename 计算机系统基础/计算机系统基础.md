
# 二、x86-64寻址方式

## 1 立即寻址（Immediate Addressing）

### 概念

操作数 **直接包含在指令中**，CPU 不需要再访问寄存器或内存。

### 示例

```asm
mov rax, 5
add rbx, 10
```

### 说明

- 操作数是常数
    
- 不需要访问内存
    
- 执行速度快
    

### 特点

|项目|内容|
|---|---|
|操作数位置|指令内部|
|是否访问内存|否|
|速度|快|


## 2 寄存器寻址（Register Addressing）

### 概念

操作数 **存放在寄存器中**。

### 示例

```asm
mov rax, rbx
add rax, rcx
```

### 说明

CPU 直接从寄存器读取数据。

### 特点

|项目|内容|
|---|---|
|操作数位置|CPU寄存器|
|是否访问内存|否|
|速度|最快|


## 3 直接寻址（Direct Addressing）

### 概念

**指令中直接给出操作数所在的内存地址**。

### 示例

```asm
mov rax, [0x600000]
```

### 地址计算

```
EA = 指令中的地址
```

EA = Effective Address（有效地址）

### 特点

|项目|内容|
|---|---|
|地址来源|指令|
|是否访问内存|是|
|使用情况|较少|


## 4 寄存器间接寻址（Register Indirect Addressing）

### 概念

**寄存器中存放的是内存地址**。

CPU 通过寄存器中的地址访问内存。

### 示例

```asm
mov rax, [rbx]
```

### 地址计算

```
EA = (RBX)
```

含义：

```
rax = Memory[rbx]
```

### 特点

|项目|内容|
|---|---|
|地址来源|寄存器|
|是否访问内存|是|
|使用情况|非常常见|


## 5 基址寻址（Base Addressing）

### 概念

使用 **基址寄存器 + 偏移量** 来计算内存地址。

### 地址计算

```
EA = Base + Displacement
```

### 示例

```asm
mov rax, [rbx + 8]
```

### 含义

```
rax = Memory[rbx + 8]
```

### 使用场景

常用于：

- 结构体访问
    
- 栈变量访问
    
- 数据结构
    

## 6 变址寻址（Indexed Addressing）

### 概念

使用 **变址寄存器 + 位移量** 来计算地址。

### 地址计算

```
EA = Index + Displacement
```

### 示例

```asm
mov rax, [rsi + 16]
```

### 使用场景

主要用于 **数组访问**。

例如：

```c
a[i]
```


## 7 基址变址寻址（Base + Index Addressing）

### 概念

地址由 **基址寄存器和变址寄存器共同计算**。

### 地址计算

```
EA = Base + Index × Scale + Displacement
```

其中

|参数|含义|
|---|---|
|Base|基址寄存器|
|Index|变址寄存器|
|Scale|比例因子 (1,2,4,8)|
|Displacement|位移|

### 示例

```asm
mov rax, [rbx + rsi*4 + 8]
```

### 说明

```
rax = Memory[rbx + rsi*4 + 8]
```

### 典型用途

数组访问：

```c
int a[10];
a[i]
```

如果 `int = 4字节`

```
地址 = base + i × 4
```


## x86-64 内存寻址统一公式

x86 架构的内存地址几乎都可以表示为：

```
EA = Base + Index × Scale + Displacement
```

说明：

| 元素           | 是否必须    |
| ------------ | ------- |
| Base         | 可选      |
| Index        | 可选      |
| Scale        | 1/2/4/8 |
| Displacement | 可选      |

示例：

```
[rax]
[rax+8]
[rax+rbx]
[rax+rbx*4]
[rax+rbx*4+8]
```


## 总结表

|寻址方式|示例|地址计算|
|---|---|---|
|立即寻址|`mov rax,5`|操作数在指令中|
|寄存器寻址|`mov rax,rbx`|操作数在寄存器|
|直接寻址|`[0x600000]`|EA=Address|
|寄存器间接寻址|`[rbx]`|EA=(RBX)|
|基址寻址|`[rbx+8]`|Base+Disp|
|变址寻址|`[rsi+16]`|Index+Disp|
|基址变址寻址|`[rbx+rsi*4+8]`|Base+Index×Scale+Disp|
